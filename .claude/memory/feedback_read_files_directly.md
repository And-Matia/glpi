metadata:

Read files directly without asking for permission first. Just notify the user that you've looked at the file (e.g., "J'ai regardé auth.service.ts").

**Why:** User finds it more efficient to have Claude read files proactively and report findings, rather than asking each time.

**How to apply:** Whenever you need to inspect a file during a task, read it directly and mention it briefly before sharing findings.
